# php-fd-scanner
PHP File/Directory Scanner

`require('fdscanner.class.php');
$fd = new FDScanner;`

**Directory List**

`$fd->list();`

**Read One File**

`$fd->oneReadFile();`

**Read Many File**

`$fd->manyReadFile();`

**Search File Name**

`$fd->searchFileName('file');`

**Search File Name Extentions**

`$fd->searchFileNameExtentions('txt');`
